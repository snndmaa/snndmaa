### 🛠️ Actix Web and Swagger Integration

**Actix Web** enables efficient handling of HTTP requests for high-performance Rust applications. Combined with **Swagger**, this project offers:

- **API Documentation:** Automatically generated API docs, making it easy to test endpoints.
- **Type-Safe API:** Type-checked route handlers and validation using Rust’s strict typing system.
  
For setup:
1. Install Actix Web and Swagger in your Rust project.
2. Set up Swagger docs in `main.rs` with the API routes.
3. Integrate the Swagger UI to visualize and test endpoints.
